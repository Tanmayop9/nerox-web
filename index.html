Commands

All available commands at your fingertips

/play

Play a song from YouTube, Spotify, SoundCloud, or Apple Music

Usage: /play [song name or URL]
Example: /play Never Gonna Give You Up
Aliases: p
/pause

Pause the currently playing track without removing it from queue

Usage: /pause
Aliases: pa
/resume

Resume playback of the paused track from where it stopped

Usage: /resume
Aliases: r, unpause
/skip

Skip the current song and play the next track in the queue

Usage: /skip [amount]
Example: /skip 3 (skips 3 songs)
Aliases: s, next
/previous

Go back and play the previous song from history

Usage: /previous
Aliases: prev, back
/stop

Stop playback completely and clear the entire queue

Usage: /stop
Note: This will disconnect the bot from the voice channel
/queue

Display the current music queue with all upcoming tracks

Usage: /queue [page]
Example: /queue 2
Aliases: q, list
/nowplaying

Show detailed information about the currently playing track

Usage: /nowplaying
Aliases: np, current
Shows: Title, artist, duration, progress bar, requester
/volume

Adjust the playback volume level (0-200%)

Usage: /volume [0-200]
Example: /volume 80
Aliases: vol, v
/shuffle

Randomly shuffle all songs in the current queue

Usage: /shuffle
Aliases: mix, randomize
/clear

Remove all songs from the queue (keeps current song playing)

Usage: /clear
Aliases: clearqueue, cq
/seek

Jump to a specific timestamp in the current track

Usage: /seek [time]
Example: /seek 1:30 or /seek 90 (seconds)
Formats: MM:SS, HH:MM:SS, or seconds
/loop

Loop the current track, queue, or disable looping

Usage: /loop [song/queue/off]
Example: /loop song
Aliases: repeat
/autoplay

Automatically play similar songs when queue ends

Usage: /autoplay [on/off]
Aliases: ap
/247

Keep the bot in your voice channel 24/7 for instant access

Usage: /247 [on/off]
Note: Bot stays connected even when idle
/join

Make the bot join your current voice channel

Usage: /join
Aliases: connect, summon
/leave

Make the bot leave the voice channel and stop playback

Usage: /leave
Aliases: disconnect, dc
/radio

Stream live radio stations from around the world

Usage: /radio [station name]
Example: /radio lofi
Stations: lofi, chill, rock, pop, jazz, classical
/similar

Play songs similar to the currently playing track

Usage: /similar
Aliases: related
/filters

Apply audio filters like bass boost, nightcore, vaporwave

Usage: /filters [filter name]
Example: /filters bassboost
Available: bassboost, nightcore, vaporwave, 8d, earrape, off
/lyrics

Display lyrics for the currently playing song or search

Usage: /lyrics [song name]
Example: /lyrics bohemian rhapsody
Aliases: ly
/remove

Remove a specific song from the queue by position

Usage: /remove [position]
Example: /remove 5
Aliases: rm
/move

Move a song to a different position in the queue

Usage: /move [from] [to]
Example: /move 5 2
/search

Search for songs and choose from top results

Usage: /search [song name]
Example: /search imagine dragons
Shows: Top 10 results to choose from
/help

Display all available commands with descriptions and usage

Usage: /help [command name]
Example: /help play
Aliases: h, commands
/ping

Check the bot's latency and response time to Discord API

Usage: /ping
Shows: Bot latency, API latency, uptime
Aliases: latency
/botinfo

Display detailed information about Nerox bot

Usage: /botinfo
Shows: Version, servers, users, uptime, memory, commands
Aliases: about, info
/serverinfo

Display detailed information about the current Discord server

Usage: /serverinfo
Shows: Owner, members, channels, roles, boost level, created date
Aliases: server, guild
/avatar

Display the avatar of yourself or another user in high quality

Usage: /avatar [@user]
Example: /avatar @Tanmay
Aliases: av, pfp
/profile

View your or another user's music profile and listening statistics

Usage: /profile [@user]
Shows: Songs played, time listened, favorite genres, top tracks
Aliases: userinfo
/stats

Show comprehensive bot statistics and performance metrics

Usage: /stats
Shows: Total servers, users, songs played, uptime, system info
Aliases: statistics
/invite

Get the invite link to add Nerox to your server

Usage: /invite
Also shows: Support server link, website link
/support

Get the link to join our support Discord server

Usage: /support
Get help: Bug reports, feature requests, general support
/vote

Vote for Nerox on top bot lists to support development

Usage: /vote
Rewards: Get premium features for 12 hours after voting!
/language

Change the bot's language for your server

Usage: /language [code]
Example: /language es
Available: en, es, fr, de, pt, ru, ja, ko
/like

Add the currently playing song to your personal favorites collection

Usage: /like
Note: Likes are saved per user, accessible from any server
Aliases: favorite, fav
/unlike

Remove a specific song from your liked songs collection

Usage: /unlike [song name or position]
Example: /unlike 5
Aliases: unfavorite, unfav
/liked

Display your complete list of liked songs with pagination

Usage: /liked [page]
Example: /liked 2
Shows: Song title, artist, duration, date added
Aliases: favorites, favs
/playliked

Play all your liked songs in order or shuffled

Usage: /playliked [shuffle]
Example: /playliked shuffle
Aliases: playfavorites, playfavs
/clearlikes

Remove all songs from your liked collection (requires confirmation)

Usage: /clearlikes
Warning: This action cannot be undone!
Aliases: clearfavorites
/importplaylist

Import a Spotify or YouTube playlist to your liked songs

Usage: /importplaylist [URL]
Example: /importplaylist [spotify link]
Supports: Spotify, YouTube, SoundCloud playlists
/exportplaylist

Export your liked songs as a shareable playlist link

Usage: /exportplaylist
Creates: A shareable link that others can import
/setup

Run the interactive setup wizard to configure Nerox for your server

Usage: /setup
Permission: Requires Administrator or Manage Server
Configures: DJ role, announcements, permissions
/djrole

Set a DJ role that can control music without restrictions

Usage: /djrole [@role]
Example: /djrole @DJ
Permission: Requires Administrator
/announce

Set a channel for music announcements (now playing, queue updates)

Usage: /announce [#channel]
Example: /announce #music
Permission: Requires Manage Server
/defaultvolume

Set the default volume level when bot joins (0-200%)

Usage: /defaultvolume [0-200]
Example: /defaultvolume 75
Permission: Requires Manage Server
/voteskip

Configure vote skip settings (enable/disable, percentage required)

Usage: /voteskip [on/off] [percentage]
Example: /voteskip on 60
Permission: Requires Manage Server
/restrictions

Set restrictions on song duration, queue size, and more

Usage: /restrictions [type] [value]
Example: /restrictions maxduration 10m
Permission: Requires Administrator
/reset

Reset all server settings to default values

Usage: /reset [confirm]
Warning: This will reset all custom configurations
Permission: Requires Administrator
/settings

View all current server settings and configurations

Usage: /settings
Shows: All configured options for your server
Aliases: config

Everything you need to know about Nerox
