Commands

All available commands at your fingertips

play

Play a song from YouTube, Spotify, or SoundCloud

pause

Pause the current track

resume

Resume the paused track

skip

Skip to the next song in queue

previous

Play the previous song

stop

Stop playback and clear queue

queue

Display the current queue

nowplaying

Show currently playing track

volume

Adjust the playback volume

shuffle

Shuffle the queue

clear

Clear the entire queue

seek

Seek to a specific time in the track

autoplay

Toggle autoplay mode

247

Enable 24/7 mode

join

Join your voice channel

leave

Leave the voice channel

radio

Play radio stations

similar

Play similar songs

enhance

Enhance audio quality

help

Display all available commands

ping

Check bot latency

botinfo

Display bot information

serverinfo

Display server information

avatar

Show user's avatar

profile

View user profile and stats

stats

Show bot statistics

invite

Get bot invite link

config

View server configuration

meta

Show metadata information

redeem

Redeem premium codes

like

Like the current playing song

unlike

Remove a song from liked list

showliked

Display your liked songs

playliked

Play your liked songs

clearlikes

Clear all liked songs

prefix

Change server prefix
