# Avatar Images

This directory contains the avatar images for the Nerox website.

## Required Images

Please download and place the following images in this directory:

1. **nerox-logo-64.gif** - Nerox bot avatar (64x64)
   - Source: https://cdn.discordapp.com/avatars/1346834405074796545/a_28c9247a1eb91089102ceb054f4c09f2.gif?size=64

2. **nerox-bot-512.gif** - Nerox bot avatar large (512x512)
   - Source: https://cdn.discordapp.com/avatars/1346834405074796545/a_28c9247a1eb91089102ceb054f4c09f2.gif?size=512

3. **casino-avatar-256.webp** - CASINO's avatar (256x256)
   - Source: https://cdn.discordapp.com/avatars/1056087251068649522/91ac78b7e3c1df4dd8e17020fea3f780.webp?size=256

4. **tanmay-avatar-256.webp** - Tanmay's avatar (256x256)
   - Source: https://cdn.discordapp.com/avatars/991517803700027443/3dcce178d4d53158b8cc9ee444c60db8.webp?size=256

## How to Download

You can download these images using curl or wget:

```bash
cd images
curl -L "https://cdn.discordapp.com/avatars/1346834405074796545/a_28c9247a1eb91089102ceb054f4c09f2.gif?size=64" -o nerox-logo-64.gif
curl -L "https://cdn.discordapp.com/avatars/1346834405074796545/a_28c9247a1eb91089102ceb054f4c09f2.gif?size=512" -o nerox-bot-512.gif
curl -L "https://cdn.discordapp.com/avatars/1056087251068649522/91ac78b7e3c1df4dd8e17020fea3f780.webp?size=256" -o casino-avatar-256.webp
curl -L "https://cdn.discordapp.com/avatars/991517803700027443/3dcce178d4d53158b8cc9ee444c60db8.webp?size=256" -o tanmay-avatar-256.webp
```

Or visit the URLs in your browser and save the images manually.
