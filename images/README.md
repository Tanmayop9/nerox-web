# Avatar Images

This directory contains the avatar images for the Nerox website.

## Available Images

The following images are included in this directory:

1. **nerox-logo.gif** - Nerox bot animated logo (64x64)
   - Used in: Navigation bar, Footer

2. **nerox-avatar.gif** - Nerox bot animated avatar (414x414)
   - Used in: Hero section, README banner

3. **casino-avatar.webp** - CASINO's avatar (256x256)
   - Used in: Team section

4. **tanmay.webp** - Tanmay's avatar (256x256)
   - Used in: Team section

All images are locally hosted and no longer rely on Discord CDN links.
