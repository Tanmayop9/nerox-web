## ğŸ“¦ Deployment

### GitHub Pages (Automatic)

This repository is configured with GitHub Actions for automatic deployment:

1. Push changes to the `main` or `master` branch
2. GitHub Actions will automatically deploy to GitHub Pages
3. Visit your site at `https://yourusername.github.io/nerox-web/`

See [.github/workflows/README.md](.github/workflows/README.md) for detailed workflow documentation.

### Custom Server (FTP)

To deploy to nerox.novacloud.tech or any custom server:

1. Set up FTP secrets in repository settings
2. Push to main/master branch
3. The FTP workflow will automatically deploy

See [DEPLOYMENT.md](DEPLOYMENT.md) for all deployment options.
